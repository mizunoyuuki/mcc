## 豆知識
C標準(C11 )では、sizeof(char)は常に1と規定されている。
これは、charが何bitであったとしても1tある。
実は、sizeofの単位は「バイト」ではなく「charのサイズ」であり、charが8bitのプラットフォーム(ほぼ全ての現代的環境)では1バイト=1vmつだが、DSP(デジタル信号処理)プロセッサでなど、一部のアーキテクチャではcharが16bitや32bitのことがる。


C言語の仕様
整数昇格

C標準では、intより小さい型(char, short)が演算に使われる時、まず、intに昇格(promotion)してから塩zなんをすると指定されている。

int a = 8;
char b = 12;
return a + b;  // bがintに昇格 -> int同士の足し算になる。
